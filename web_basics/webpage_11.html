Absolute Head



CSS positioning is all about, positioning the elements in all the possible direction. Not just, providing the position it also have extra functionality.

Absolute postioning is done with respect to the main html page. Positioning is done through the properties top,left right,down.

Relative Head

Relative postioning is done (w.r.t) the parent container it is present in. If their is no parent container defined html page is used for postioning.

Sticky Head

sticky postioned elements have extra function like, it works as a normal relative positioning until it's top position reached and sticks their.

Fixed Head

Fixed positioning will fix the particular element in a particular position defined even if the we scroll the document the fixed postion element just lies their
