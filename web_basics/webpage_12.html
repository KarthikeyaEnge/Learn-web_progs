Hello, How are you guys!!. Today the topic is FlexBox Layout. We have seen how to arrange the elements in the page using display, position properties and it's values.


Even doing positioning using above mentioned properties is ok, but they are more tedious, Flex box layout is more flexible and eay to implement and use in web designing.


Flex box has various attributes like....

  • Justify-contents
  • Align-items
  • Align-contents
  • Align-self
  • Flex-direction
  • Flex-wrap
  • Flex-flow
  • Order
  • Flex-grow
  • Flex-basis
  • Flex

JUSTIFY-CONTENTS


It is used to align items in the main axis either horizontal or vertical, also used to provide space between the elements evenly, etc.;

values: flex-end, flex-start, center, space-between, space-around, space evenly.


ALIGN-ITEMS


It is used to align the items in the cross axis. It's opposite to the justify-contents

values: flex-start, flex-end, center, stretch, baseline.


FLEX-DIRECTION


It is used to define the direction of the elements as row or column also used to reverse the elements.

values: row, row-reverse, column, column-reverse.


FLEX-WRAP


It is used to wrap the elements when the screen size shrinks.

values: wrap, nowrap, wrap-reverse.


FLEX-FLOW


It is seen that both the flex-direction and flex-wrap are used side by side. So, flex flow is used to define both flex-direction & flex-wrap.

syntax: flex-flow: flex-direction flex-wrap;


ALIGN-CONTENTS


It is used to align the elements inside the container itself.values same as the align-items.
